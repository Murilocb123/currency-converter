# Currency Converter

## Trabalho 02 - Programação Funcional

# Como usar
- Requisitos:
    - Node: v20.17.0

- Inicializando o projeto:
    - Clone o repositório e execute linha a linha:
    ```bash	
    > git clone {url}
    > npm install
    > npm start
    ``` 
- Acesse o projeto em: http://localhost:4200/
